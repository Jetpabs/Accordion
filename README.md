# Accordion
A simple FAQ accordion component based on the roadmap.sh project to practice HTML, CSS, and JavaScript.

**Link to the webpage:** https://jetpabs.github.io/Accordion/

**Link to roadmap.sh project:** https://roadmap.sh/projects/accordion

## How It's Made
**Tech used:** HTML, CSS, JavaScript, Heroku

This project is a simple practice implementation of an accordion-style FAQ component. It demonstrates basic DOM manipulation and styling techniques.


## Lessons Learned
This project served as a refresher for building interactive webpages using HTML structure, CSS styling, and JavaScript event handling.
